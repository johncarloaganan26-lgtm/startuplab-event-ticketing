# Technology Stack

## Programming Languages

### Backend
- **JavaScript (ES Modules)** - Node.js runtime with ES6+ features
- **SQL** - PostgreSQL database queries and migrations

### Frontend
- **TypeScript 5.8.2** - Strict type checking enabled
- **JSX/TSX** - React component syntax

### Utilities
- **Python 3.x** - Build scripts and automation tools

## Core Frameworks & Libraries

### Backend Stack
- **Express 5.2.1** - Web application framework
- **Node.js** - JavaScript runtime (ES Module type)
- **Supabase JS 2.93.3** - PostgreSQL client and authentication
- **Nodemailer 7.0.13** - Email sending service
- **JWT Decode 4.0.0** - Token authentication

### Frontend Stack
- **React 19.2.4** - UI component library
- **React DOM 19.2.4** - React rendering
- **React Router DOM 7.13.0** - Client-side routing
- **Vite 6.2.0** - Build tool and dev server
- **Supabase JS 2.93.3** - Database and auth client

## Database & Storage

### Primary Database
- **Supabase (PostgreSQL)** - Relational database with real-time capabilities
- **Schema Management** - SQL migration files with timestamps
- **Authentication** - Supabase Auth for user management

### Key Tables
- Users, Organizers, Events, Ticket Types, Orders, Tickets
- Subscriptions, Plans, Promotions
- Notifications, Support Messages, Audit Logs
- Email Quota Tracking, User Settings

## External Services & APIs

### Payment Processing
- **HitPay** - Payment gateway with webhook integration
- Webhook endpoint for payment confirmation
- Order status tracking

### Email Services
- **System SMTP** - Default email delivery
- **Custom SMTP** - Professional/Enterprise plan feature
- **Nodemailer** - Email client library

### Authentication
- **Supabase Auth** - JWT-based authentication
- Cookie-based session management
- Role-based access control (user, organizer, admin)

## Development Tools

### Build Systems
- **Vite** - Frontend bundler with HMR (Hot Module Replacement)
- **TypeScript Compiler** - Type checking and transpilation
- **ES Modules** - Native JavaScript module system

### Development Servers
- **Vite Dev Server** - Frontend development (port 5173 default)
- **Nodemon** - Backend auto-reload on file changes
- **Express Server** - API server

### Code Quality
- **TypeScript Strict Mode** - Enhanced type safety
- **ES6+ Features** - Modern JavaScript syntax
- **Module System** - ES Modules throughout

## Environment Configuration

### Backend Environment Variables
```
# Database
SUPABASE_URL=
SUPABASE_KEY=
SUPABASE_SERVICE_ROLE_KEY=

# Payment Gateway
HITPAY_API_KEY=
HITPAY_SALT=
HITPAY_WEBHOOK_URL=

# Email
SMTP_HOST=
SMTP_PORT=
SMTP_USER=
SMTP_PASS=

# Application
JWT_SECRET=
NODE_ENV=development|production
PORT=3000
```

### Frontend Environment Variables
```
VITE_API_URL=http://localhost:3000
VITE_SUPABASE_URL=
VITE_SUPABASE_ANON_KEY=
```

## Development Commands

### Backend
```bash
npm run dev          # Start development server with nodemon
npm test            # Run tests (placeholder)
node server.js      # Start production server
```

### Frontend
```bash
npm run dev         # Start Vite dev server
npm run build       # Build for production
npm run preview     # Preview production build
```

### Root Level
```bash
npm install         # Install shared dependencies (date-fns, QR libraries)
```
